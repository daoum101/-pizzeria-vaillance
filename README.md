Structure prête pour GitHub Pages.

## Fichiers inclus
- index.html
- styles.css
- script.js
- i18n.js
- manifest.json
- images optimisées : hero, terrasse, intérieur, façade
- icônes PWA 192/512

## Mise en ligne GitHub Pages
1. Ouvrir le repository GitHub.
2. Cliquer sur Add file → Upload files.
3. Envoyer tous les fichiers du ZIP directement à la racine.
4. Ne pas mettre dans un dossier `/site`.
5. Settings → Pages → Deploy from branch → main / root.
